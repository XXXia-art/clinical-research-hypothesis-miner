# 科研决策卡模板

本文件用于 Phase 7。它接收 Phase 3-6 的阶段产物，将文献与资源证据概述、创新点证据地图、研究空白审计结果、可检验假说和验证路径整理成面向用户的最终科研决策卡。

Phase 7 不重新检索，不重新绘制证据地图，不重新生成假说。它只负责压缩展示已有阶段产物，并给出 Start/Narrow/Redesign/Stop 最终决策。

除非用户要求其他格式，最终回答使用以下结构。简单任务可以压缩小节，但不得省略来源范围、证据边界、空白审计、假说、验证路径和最终决策。

## 一、问题与来源范围

| 项目 | 内容 |
|---|---|
| 原始问题 | 用户原始表述 |
| 标准化研究问题 | Phase 1 生成的中文科研问题 |
| English retrieval question | Phase 1 生成的英文检索问题 |
| 关键实体与边界 | 疾病、实体、终点、研究对象、排除边界 |
| 来源范围 | PubMed、arXiv、Web/Tavily、用户提供材料的实际状态 |

如果没有执行实时检索，必须写明：本次未执行实时检索，以下内容只能作为检索计划、证据整理框架或研究设计建议，不能当作已完成的系统综述结论。

| 假说 | 假说类型 | 证据锚点 | 假说逻辑链 | 关键读出 | 最小验证 | 失败判据 |
|---|---|---|---|---|---|---|

使用 1-3 个假说。若当前证据不足以形成可靠假说，说明需要补证据、收窄方向或重构问题，不强行生成。

## 六、验证路径

使用 Phase 6 的验证阶梯，基于 Phase 2-6 已有证据和用户可用资源设计下一步，不在 Phase 7 新增检索任务。

| 版本 | 适用条件 | 核心工作 | 产出 |
|---|---|---|---|
| Lite | 只有文献、已有证据包或公共数据线索 | 基于已有证据包和可获得公共数据，规划表达、通路、生存、相关性或初步证据图分析 | protocol seed、初步证据图或待补证据清单 |
| Standard | 有临床样本或基础实验条件 | 本地样本验证、qPCR/IHC/ELISA/western blot、流式、简单功能实验 | 原创研究路径或小规模验证结果 |
| Publication+ | 有更强协作和平台资源 | 独立队列、多组学、单细胞/空间、动物或 rescue、外部验证 | 更强机制或转化研究路径 |

推荐其中一个版本作为下一步，并说明为什么该版本最匹配当前证据和资源。

## 七、关键风险与待补证据

汇总 Phase 3-6 中仍会影响最终决策的风险，不重复所有阶段细节。

| 风险或缺口 | 来自阶段 | 影响 | 建议处理 |
|---|---|---|---|

常见来源包括：证据来源未获得、证据层级较低、结论冲突、因果过度、转化过度、数据/样本不可得、验证负担过重。

## 八、评分与最终决策

评分用于辅助决策，不替代文字判断。若证据不足以评分，可写 `无法可靠评分`，并说明原因。

| 维度 | 分数 | 理由 |
|---|---:|---|
| 科学问题清晰度 | /20 |  |
| 文献/证据支撑 | /20 |  |
| 研究空白可信度 | /20 |  |
| 假说可检验性 | /15 |  |
| 数据/样本/实验可行性 | /15 |  |
| 发表或转化潜力 | /10 |  |
| **Total** | **/100** |  |

最终决策：`Start / Narrow / Redesign / Stop`。

决策含义：

- **Start：** 证据锚点清楚，研究空白可信，假说可检验，最小验证路径现实。
- **Narrow：** 方向有价值，但问题边界、对象、终点或验证路径需要收窄。
- **Redesign：** 当前想法有潜力，但证据链、假说结构或资源条件需要重构。
- **Stop：** 当前方向主要是伪空白、证据不足、过度宣称或现实不可行。
