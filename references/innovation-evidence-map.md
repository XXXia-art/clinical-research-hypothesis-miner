# 创新点证据地图

本文件用于 Phase 4。它接收 Phase 3 生成的文献与资源证据概述，把已有证据分布转化为候选创新方向地图。

Phase 4 的任务不是最终判断课题是否值得启动，也不是生成假说或验证方案，而是回答：哪些方向已经拥挤，哪些方向薄弱，哪些方向存在冲突，哪些方向可能形成后续审计对象。

## 使用时机

仅在完成 Phase 3 的文献与资源证据概述后使用。如果证据概述未完成、来源状态未标注，或核心材料状态为 `未获得`，不得进入创新点证据地图。

## Phase 4 边界

Phase 4 只做“地图层”判断：

- 可以描述候选方向的证据密度、证据层级、机制链完整度、争议状态和来源线索。
- 可以标注某个方向“可能拥挤”“可能薄弱”“可能存在真实机会”。
- 可以指出疑似伪空白，但最终剔除交给 Phase 5。
- 可以指出后续需要验证的入口，但具体假说、实验路径和验证阶梯交给 Phase 6。
- 不输出 Start / Narrow / Redesign / Stop 决策；最终决策交给 Phase 7。

## 输入材料

Phase 4 只接收 Phase 3 已整理出的内容：

| 输入 | 用途 |
|---|---|
| 文献与资源分布 | 判断哪些主题、模型、方法和终点已经被覆盖 |
| 来源贡献与边界 | 判断每类来源能支持什么、不能支持什么 |
| 代表性记录 | 作为候选方向的证据锚点 |
| 待补证据 | 标注证据不足或不能判断的方向 |

不得在 Phase 4 重新检索、补造记录、补造 PMID/DOI、补造数据集或补造试验注册信息。

## 候选方向识别

先把 Phase 3 的概述拆成若干候选方向。候选方向应来自证据材料，而不是从常见套路中硬造。

优先识别四类方向：

- **拥挤方向：** 直接证据多，研究问题已经高度重复，适合作为背景，不适合作为主要创新点。
- **薄弱方向：** 证据稀疏、层级较低或验证不足，但尚不能直接宣称是真实空白。
- **冲突方向：** 结果不一致，且冲突可能来自人群、模型、终点、时间点、样本来源或方法差异。
- **可审计机会：** 有明确证据锚点和可解释缺口，值得交给 Phase 5 做真实空白审计。

## 创新点证据地图

用下表整理候选方向：

| 候选方向 | 当前证据图景 | 证据密度 | 证据层级 | 机制链 | 冲突/争议 | 来源线索 | 初步地图判断 | 下一阶段处理 |
|---|---|---|---|---|---|---|---|---|

字段定义：

- **当前证据图景：** 简述该方向在 Phase 3 中呈现出的覆盖情况。
- **证据密度：** dense / moderate / thin / unknown。
- **证据层级：** review、human、public omics、cell、animal、trial、database、web、user-provided。
- **机制链：** complete / partial / broken / speculative / unknown。
- **冲突/争议：** none / explainable / unresolved / high / unknown。
- **来源线索：** PubMed evidence、arXiv method、official web、user-provided、not available。
- **初步地图判断：** crowded、thin-area、conflict-area、candidate-opportunity、insufficient-evidence。
- **下一阶段处理：** enter Phase 5 audit / keep as background / mark as evidence-limited / not applicable。

## 缺口类型标注

对 `candidate-opportunity` 或 `conflict-area`，可以标注可能的缺口类型，但不要在本阶段确认其为真实空白。

| 缺口类型 | 含义 | Phase 4 表述方式 |
|---|---|---|
| 机制链缺口 | 疾病 -> 分子事件 -> 细胞/组织表型 -> 终点之间某一环不足 | “可能存在机制链薄弱环节，需 Phase 5 审计” |
| 证据层级缺口 | 有低层级证据但缺人群、功能、外部或临床相关验证 | “证据层级可能不足，需判断是否影响结论” |
| 冲突解释缺口 | 不同研究方向或结果不一致，原因尚未解释 | “存在待解释冲突，需审计冲突来源” |
| 转化衔接缺口 | 机制线索尚未连接到 biomarker、靶点、患者分层或临床终点 | “可能存在转化衔接不足” |
| 方法分辨率缺口 | 既有方法无法回答细胞状态、亚群、空间位置或动态变化问题 | “可能需要更高分辨率方法，但不得直接等同创新” |
| 情境边界缺口 | 人群、疾病阶段、样本来源、治疗背景或模型边界不清楚 | “可能存在特定情境未覆盖” |

## 疑似伪空白标记

Phase 4 只做标记，不做最终剔除。若候选方向出现以下特征，在地图中标为 `possible pseudo-gap`，交给 Phase 5 审计：

- 只是“文献少”，但没有清楚说明科学问题为什么重要。
- 只是“加单细胞”“加多组学”“用机器学习”，但没有对应的机制、终点或临床问题。
- 只是换疾病、换数据集、扩大样本量，而没有改变科学问题或证据层级。
- 只是把相关性包装成因果、靶点或临床效用。
- 来源主要来自弱网页或未核验材料，缺少 PubMed、官方页面或用户提供材料支撑。

## Phase 4 输出顺序

除非用户要求极简回答，Phase 4 按以下顺序输出：

1. 候选方向列表。
2. 创新点证据地图表。
3. 缺口类型标注。
4. 疑似伪空白标记。
5. 交给 Phase 5 审计的候选方向。

输出末尾只说明哪些方向应进入 Phase 5，不展开正式验证方案，不输出最终启动决策。
