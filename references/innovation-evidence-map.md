# 创新点证据地图

本文件用于 Phase 4。它接收 Phase 3 生成的文献与资源证据概述，把已有证据分布转化为候选创新方向地图。

Phase 4 的任务不是最终判断课题是否值得启动，也不是生成假说或验证方案，而是回答：当前证据图景中有哪些方向已经拥挤、哪些方向薄弱、哪些方向存在冲突、哪些方向呈现出候选创新可能。

## 启动条件

仅在完成 Phase 3 的文献与资源证据概述后使用。如果证据概述未完成、来源状态未标注，或核心材料状态为 `未获得`，不得进入创新点证据地图。

## 输入材料

Phase 4 只接收 Phase 3 已整理出的内容：

| 输入 | 用途 |
|---|---|
| 文献与资源分布 | 判断哪些主题、模型、方法和终点已经被覆盖 |
| 来源贡献与边界 | 判断每类来源能支持什么、不能支持什么 |
| 代表性记录 | 作为候选方向的证据锚点 |
| 待补证据 | 标注证据不足或不能判断的方向 |

不得在 Phase 4 重新检索、补造记录、补造 PMID/DOI、补造数据集或补造试验注册信息。

## 方向类型

- **拥挤方向：** 直接证据多，研究问题已经高度重复，适合作为背景，不适合作为主要创新点。
- **薄弱方向：** 证据稀疏、层级较低或验证不足，只能作为薄弱区域记录。
- **冲突方向：** 结果不一致，且冲突可能来自人群、模型、终点、时间点、样本来源或方法差异。
- **候选机会：** 有明确证据锚点和可解释薄弱点，可能形成后续创新判断的材料。

每个方向只归入一个主类型。若同一方向同时符合多个类型，选择最能解释证据图景的一个，不额外叠加多个标签。

## 地图表格

用下表整理候选方向：

| 候选方向 | 主类型 | 主要证据依据 | 候选机会说明 |
|---|---|---|---|

字段定义：

- **主类型：** 拥挤方向 / 薄弱方向 / 冲突方向 / 候选机会 / 证据不足。
- **主要证据依据：** 用 Phase 3 中的代表性记录、来源类型或材料范围概括，不堆叠多层标签。
- **候选机会说明：** 仅当主类型为候选机会时填写；其他类型可写 `不适用`。

## 输出顺序

除非用户要求极简回答，Phase 4 按以下顺序输出：

1. 创新点证据地图表。
2. 候选机会小结。

创新点证据地图应参考上文章节地图表格，输出末尾只总结 Phase 4 观察到的候选机会和不确定性，不展开正式验证方案，不输出最终启动决策。
