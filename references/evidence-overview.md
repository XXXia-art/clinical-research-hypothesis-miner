# 文献与资源证据概述

本文件用于 Phase 3。它接收 Phase 2 已标准化返回记录表，概述当前材料覆盖的研究主题、文献与资源分布、不同来源的贡献边界。Phase 3 不重新检索，不补造缺失字段，不判断创新点，不生成假说，也不做后续决策评分或最终取舍；所有数量、趋势、代表性记录和资源线索都必须能回到 Phase 2 的返回记录表。

## 输入要求

Phase 3 的输入优先来自 Phase 2 生成的 `evidence-packet-[topic-slug]-[YYYYMMDD].md` 文件级记录，否则依托于Phase 2 标准化的材料，如果没有该文件或同结构内容，Phase 3 不应继续生成文献与资源概述。

Phase 3 只接收已经过 Phase 2 标准化的材料：

| 输入 | 来源 | 用途 |
|---|---|---|
| PubMed evidence packet | Phase 2 | 建立核心医学文献主干 |
| arXiv methods packet | Phase 2 | 补充 AI、统计、多组学、计算方法和预印本方法线索 |
| Web verification packet | Phase 2 | 核验数据集、临床试验、指南、数据库入口、官方页面和论文线索 |
| 用户材料包 | Phase 2 | 保留用户实际提供的论文、摘要、实验结果、笔记或导出材料 |

## 证据完整性规则

在生成概述前，先确认每类材料的状态、字段和边界。该规则统一处理“未获得”“不适用”“弱来源”和缺失字段，避免在后文重复说明。

| 情况 | Phase 3 处理 | 不得做什么 |
|---|---|---|
| 来源状态为 `未获得` | 放入待补证据，说明缺口和需要补什么 | 不得写成已获得证据 |
| 来源状态为 `不适用` | 简要说明为什么本轮不需要该来源 | 不得为了凑齐三源而强行补检索 |
| 字段缺失 | 将缺少的 PMID、DOI、年份、期刊、URL、摘要等标注为 `未提供` | 不得凭记忆或常识补全 |
| 弱来源或普通网页 | 保留 Phase 2 的弱来源标注，只作为线索 | 不得升级为正式医学证据 |
| 用户材料 | 标注为用户提供材料，并说明覆盖范围 | 不得声称代表全领域系统检索结果 |

若来源没有实际返回结果，不要编造代表性文献、数据集、试验编号、指南结论、引用次数或 landmark 地位。

## 文献与资源分布

Phase 3 可以做描述性概述，但不要做创新性判断。所有概述都应来自 Phase 2 的记录字段，例如 PubMed 的 `published`、`journal`、`pub_types`、`keywords`、`mesh_terms`、`abstract`，arXiv 的 `title`、`abstract`，以及 Web/Tavily 的 `title`、`snippet`、`domain`。

| 维度 | 需要报告 |
|---|---|
| 年份分布 | 从 `published` 提取，不得只因返回结果较新就写“领域高度活跃” |
| 研究类型 | 优先使用 PubMed `pub_types`；没有字段时写 `当前材料不足以判断` |
| 疾病和模型覆盖 | 仅从标题、摘要、MeSH、关键词或用户材料中提取 |
| 方法与数据资源 | 仅记录材料中实际出现的方法、数据集、模型或资源；未由 Web/Tavily 核验的资源标为待核验 |
| 终点 | 仅记录材料中实际出现的诊断、预后、生存、治疗反应、分子表型等终点 |
| 主题词 | 优先来自 `keywords`、`mesh_terms`、标题和摘要中的高频主题 |

如果材料不足以判断某个维度，写 `当前材料不足以判断`。

## 来源贡献与边界

不同来源只能支持不同层级的结论。Phase 3 需要说明贡献，也要说明边界。

| 来源类型 | 可以支持 | 不能支持 |
|---|---|---|
| PubMed 核心医学文献 | 医学背景、机制线索、临床相关性、代表性研究、已有研究密度 | 不自动证明因果、疗效、创新性或可发表性 |
| arXiv 方法学与预印本 | 方法路线、算法线索、多组学或 AI 可迁移思路 | 不能替代医学证据，不能单独支持临床或机制结论 |
| Web/Tavily 网页与数据库核验 | 数据集、试验注册、指南、官方页面、数据库入口、论文线索补查 | 不能替代正式文献证据，弱来源只能作为线索 |
| 用户提供材料 | 本轮用户实际语料中的主题、结论、实验结果或限制 | 不能代表全领域系统检索结果 |

## Phase 3 输出顺序

除非用户要求极简回答，Phase 3 按以下顺序输出：

1. 材料范围与来源状态。
2. 文献与资源分布。
3. 来源贡献与边界。
4. 待补证据。

各部分字段应以上文对应章节为准，不在此处重复表格。
