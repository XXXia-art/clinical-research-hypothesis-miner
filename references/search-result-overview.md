# 搜索结果概述

当用户提供文献检索结果，或需要把核心医学文献、方法学补充、网页核验结果整理为领域概述时，使用本参考文件。

## 输入类型

接受并标准化：

- academic-research 返回的 PubMed 综述或核心医学文献记录。
- arXiv 方法学论文或预印本结果。
- 标题和摘要。
- 从文献管理器复制出的导出文本。
- 用户已筛选论文列表。
- 混合来源笔记。

如果元数据缺失，标注为缺失。不得凭记忆补 DOI、PMID、引用次数、期刊或年份。

## 概述步骤

1. **标准化记录。** 在信息存在时提取标题、年份、期刊/来源、检索来源、摘要片段、实体、疾病、方法、模型和终点。
2. **谨慎去重。** 只合并明显重复标题；不要因为主题相似就合并记录。
3. **分类研究类型。** 综述/Meta、人群临床、公共组学、细胞实验、动物实验、临床试验、数据库/方法学论文、网页背景资料。
4. **提取主题。** 机制、细胞类型、通路、生物标志物、靶点、模型、治疗背景、终点。
5. **评估证据边界。** 说明该语料能支持什么，不能支持什么。
6. **识别检索缺口。** 指出缺失来源、缺失模型、缺失临床终点或缺失验证层级。

## 必要概述部分

### 检索范围

- 语料实际覆盖的主题。
- 涉及的来源。
- 已知时间范围。
- 结果是 top hits、用户筛选结果，还是系统检索结果。

### 文献分布

| 维度 | 需要报告 |
|---|---|
| 年份趋势 | 早期/稳定/上升/近年爆发/不清楚 |
| 研究类型 | 综述、临床、组学、细胞、动物、试验、方法学 |
| 疾病/模型覆盖 | 人群疾病阶段、组织、动物/细胞模型 |
| 方法 | qPCR、IHC、western blot、测序、单细胞、蛋白组、生信、功能实验 |
| 终点 | 诊断、预后、生存、治疗反应、分子表型 |

### 代表性文献

只从用户提供或实际检索到的材料中列出代表性记录：

| 文献 | 来源 | 年份 | 重要性 | 限制 |
|---|---|---:|---|---|

如果来源未提供年份或期刊，写 `未提供`。

### 语料能支持什么

使用克制表述：

- “支持相关性线索”。
- “提示机制可能参与”。
- “存在功能验证基础”。
- “已有转化/临床探索”。
- “目前仅能作为假说”。

### 语料不能支持什么

明确列出不能过度宣称的内容：

- 临床疗效。
- 诊断落地应用。
- 从表达相关性直接推出因果。
- 靶点成药性。
- 外推到其他人群或疾病阶段。

## 输出风格

概述要简洁但结构清楚。除非用户要求，不要写成长篇逐篇注释书目。
